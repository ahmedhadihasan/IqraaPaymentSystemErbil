# نمونەی فایلەکانی CSV بۆ سیستەمی اقرأ - لقی هەولێر

## فایلەکان

### 1. students-male-example.csv
نمونەی فایلی CSV بۆ هاوردەکردنی قوتابیانی کوڕ

### 2. students-female-example.csv
نمونەی فایلی CSV بۆ هاوردەکردنی قوتابیانی کچ

## شێوازی ستوونەکان

| ستوون | ناو | پێویست | وەسف |
|-------|-----|--------|------|
| 1 | ژمارە | نەخێر | ژمارەی ڕیز (نابینرێت) |
| 2 | ناوی سیانی | بەڵێ | ناوی تەواوی قوتابی |
| 3 | لەدایکبوون | نەخێر | ساڵی لەدایکبوون |
| 4 | ناونیشان | نەخێر | ناونیشانی قوتابی |
| 5 | ژمارەی مۆبایل | نەخێر | ژمارەی پەیوەندی |
| 6 | بارى دارایی | نەخێر | باش، ناوەند، کەم |

## ڕێنمایی
1. فایلەکە دەبێت UTF-8 بێت
2. ڕیزی یەکەم سەرناوە دەبێت
3. ڕەگەز لە سیستەم هەڵدەبژێرێت کاتی هاوردەکردن
4. تەنها ستوونی ناوی سیانی پێویستە

## نمونە
```csv
ژمارە,ناوی سیانی,لەدایکبوون,ناونیشان,ژمارەی مۆبایل,بارى دارایی
1,ئاراس محمد عبدالله,٢٠١٢,هەولێر,07501234567,باش
```
